## 📌 Naming Convention

All PYQ files follow the pattern:

📄 Subject Name (Month Year) (Course Code).pdf


For example:
- `Pollution Control and Monitoring (Dec 2023) (CLE2019).pdf`
- `Process Plant Utilities (May 2022) (CHE1010).pdf`
