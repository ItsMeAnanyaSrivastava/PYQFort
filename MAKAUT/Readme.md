## 🧾 File Naming Convention

Each PYQ file follows the format:

📄 `Course Name (Branch Name)(Semester)(Subject Name/Subject Code)(YEAR).pdf`

🔸 **Examples:**

* `btech - cse-3-sem (CH-301) (2011).pdf`
* `btech - ch-2-sem (BSM-201) (2018).pdf`
