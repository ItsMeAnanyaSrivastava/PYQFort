Contribute to PYQFort

You can contribute to PYQFort in two ways: via a simple form for quick submissions or through GitHub for technical contributions.

Option 1: Submit PYQs via Form

Easily upload your previous year question papers using our form.

Submit PYQs via Form

Fill out details like college, branch, semester, and subject, and upload your PDF or image files. Your contribution will be reviewed and added to PYQFort.

Option 2: Technical Contributions via GitHub

GitHub link: https://github.com/Saumy1905/PYQFort

ðŸ”„ Cloning the Repository on Your Laptop

To contribute, first fork the repository and then clone your fork:

git clone <your-forked-repo-URL>

ðŸ“‚ Making Changes

  1. Navigate to the cloned directory and follow the commands: 
    cd PYQFort
git remote add upstream https://github.com/Saumy1905/PYQFort.git
git remote -v
  2. Create a new branch for your contribution: 
    git checkout -b college-name
  3. Add new PYQs or make improvements.
  4. Stage the changes: 
    git add .
  5. Commit your changes with a meaningful message: 
    git commit -m "Added PYQs for [Branch] [Sem] [Subject] [Code]"
  6. Push the changes to your fork: 
    git push origin college-name

ðŸ”„ Submitting a Pull Request

  1. Go to your forked repository on GitHub.
  2. Click on "Compare & pull request".
  3. Add a brief description of your changes.
  4. Click "Create pull request".
